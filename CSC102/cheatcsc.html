CSC102: PHP Cheat Sheet

syntax · variables · control flow · arrays · forms · advanced

1. PHP Basics & Environment

  • PHP Tags <?php ... ?> – code must be inside these tags.
  • Output echo "Hello"; (multiple arguments) vs print "Hello"; (returns 1).
  • Comments // single line   # shell-style   /* multi line */
  • Statement terminator Every statement must end with ;.

2. Variables, Constants & Operators

  • Variables Always start with $, e.g. $name. Case‑sensitive. No explicit typing.
  • Scope local (inside function), global (global $var; or $GLOBALS['var']), static (retains value).
  • Constants define("NAME", "value"); or const NAME = "value"; – globally accessible.
  • Operators Arithmetic: + - * / %
    String: . (concatenation), .=
    Comparison: == (value), === (value + type), !=, <>
    Logical: && (AND), || (OR)

3. Control Flow

  • Conditionals if (condition) { … } else { … } elseif (condition) { … }
    switch ($var) { case val: …; break; }
    Ternary: $x = (cond) ? a : b;
  • Loops while (cond) { … }
    do { … } while (cond); (executes at least once)
    for ($i=0; $i<10; $i++) { … }
    foreach ($array as $key => $value) { … }
  • Jump statements break (exit loop/switch), continue (skip iteration).

4. Arrays & Built‑in Functions

  • Array types Indexed: $arr = [1,2,3];
    Associative: $arr = ["k"=>"v"];
    Multidimensional: $arr = [[1,2],[3,4]];
  • Array functions count($arr), array_sum(), array_push(), sort()
  • String functions strlen(), strtoupper(), strtolower(), substr(), strpos(), trim(), explode(), implode()
  • Math / utility rand(), round(), floor(), is_numeric(), type casting (int)$var

5. Web Interaction & Data Handling

  • Superglobals $_GET, $_POST, $_REQUEST – form data.
    $_SESSION, $_COOKIE, $_FILES, $_SERVER.
  • File inclusion include 'file.php'; (warning if missing)
    require 'file.php'; (fatal error if missing)
    include_once / require_once (prevents multiple loads)
  • Sessions & Cookies session_start(); to use $_SESSION.
    setcookie("name","value", time()+3600);
    Destroy session: session_destroy();
  • File handling fopen(), fclose(), fread(), fwrite(), unlink(), file_exists(), filesize()

6. Advanced Topics & Code Analysis

  • MySQL (legacy) mysql_connect(), mysql_query() – outdated. Use prepared statements (PDO or MySQLi) to prevent SQL injection.
  • GD Library Image manipulation: imagecreatetruecolor(), imageline(), imagerectangle(), filters, thumbnails.
  • Common output pitfalls Type coercion: echo 2 + "2"; → 4
    References: $b = &$a; makes both refer to same value
    Increment: ++$x (pre‑increment) vs $x++ (post‑increment)
    Strict comparison: === checks value and type.
