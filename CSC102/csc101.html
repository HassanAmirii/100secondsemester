Module 1: PHP Basics and Environment

  • Overview: server‑side scripting, embedded in HTML, open‑source.
  • Setup: requires web server + PHP; WampServer = Windows + Apache + MySQL + PHP.
  • Syntax & Structure:
    • Canonical tags: <?php ?>
    • Statements end with ;, whitespace ignored.
    • Comments: // single‑line, /* … */ multi‑line.
  • Output: echo (multiple arguments) vs. print (returns 1).

Module 2: Variables, Constants, and Operators

  • Variables: start with $, case‑sensitive, automatic typing.
  • Scope: local, global (global keyword, $GLOBALS array), static.
  • Constants: define('NAME', value), conventionally uppercase, global scope.
  • Operators:
    • Assignment: =
    • Arithmetic: + - * / %
    • String: concatenation . and .=
    • Comparison: ==, ===, !=, <>
    • Logical: && (AND), || (OR)
    • Precedence follows standard rules (PEMDAS).

Module 3: Control Flow

  • Branching:
    • if, else, elseif
    • switch (efficient for many values)
    • Ternary: condition ? expr1 : expr2
  • Looping:
    • while, do‑while (executes at least once)
    • for (known iteration count)
  • Jumps: break (exit loop/switch), continue (skip iteration).

Module 4: Arrays and Built‑in Functions

  • Array Types: indexed, associative, multidimensional.
  • Iteration: foreach ($array as $key => $value)
  • Array functions: count(), array_sum(), array_push(), sort()
  • String functions: strlen(), strtoupper(), substr(), strpos(), trim(), explode(), implode()
  • Math/Utility: rand(), round(), pow(), is_numeric(), type casting (int).

Module 5: Web Interaction and Data Handling

  • Superglobals & Forms: $_GET, $_POST, $_REQUEST collect form data.
  • File Inclusion: include, include_once, require, require_once.
  • State Management: Cookies (client‑side) vs. Sessions (server‑side, session_destroy()).
  • File System: fopen(), fclose(), fread(), fwrite(), unlink(), file_exists(), filesize().

Module 6: Advanced Topics and Code Analysis

  • Database (MySQL): mysql_connect(), mysql_query() (legacy), SQL injection prevention with prepared statements.
  • Image Manipulation (GD): create images, draw (imageline(), imagerectangle()), filters, thumbnails.
  • Code Output / Debugging:
    • Type coercion: echo 2 + "2" → 4
    • References: $b = &$a
    • Pre/post increment: ++$x vs $x++
    • Strict comparison: === (value + type)
