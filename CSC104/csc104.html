Module 1: Introduction to Data Analysis and Big Data

  • Core Concepts: Data analysis, data science, data science lifecycle (capture, maintain, process, analyze, communicate).
  • Data Types & Quality:
    • Structured, unstructured, semi‑structured (characteristics, advantages/disadvantages).
    • Data quality: accuracy, consistency, timeliness, completeness, relevance, validity.
  • Big Data Fundamentals:
    • Definition and examples.
    • The 4 Vs: Volume, Velocity, Variety, Veracity.
    • Evolution: from traditional processing to cloud computing and AI/ML.

Module 2: Data Modeling and Management Tools

  • Metadata: "data about data". Characteristics: descriptive, structured, contextual, relational, dynamic. Merits (enhanced understanding) and demerits (overhead).
  • Data Flows and Diagrams:
    • Data flows (internal, external, bidirectional).
    • Data Flow Diagrams (DFDs): graphical, abstraction, hierarchy. Logical vs. physical. Components: processes, data flows, data stores, external entities. Levels (0‑context, 1, 2).
    • Other diagrams: ERDs, UML class diagrams, network diagrams.
  • Databases and Structures (Weeks 6‑7): characteristics of data structures, operational databases, relational vs. non‑relational databases.

Module 3: Advanced Data Storage and Analytics Concepts

  • Data Warehouse (DW): subject‑oriented, integrated, non‑volatile, time‑variant. Architecture: data sources, ETL (extract, transform, load), staging area, data storage (ODS, data mart), metadata, access tools (query, reporting, OLAP). Benefits and challenges (scalability, data quality, integration).
  • Data Management Platforms (DMPs): role in marketing/advertising. Functions: data collection (first‑party, second‑party, third‑party), integration (unification, normalization), storage (scalability, security), segmentation.
  • Statistical Modeling: descriptive vs. inferential statistics. Forecasting techniques: predictive modeling, time series analysis.
