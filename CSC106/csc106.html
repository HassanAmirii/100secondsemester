First look at computer parts/components and tools

  • Overview of internal components: CPU, RAM, motherboard, power supply, storage drives, expansion cards.
  • Essential tools: screwdrivers (Phillips, flathead), anti‑static wrist strap, thermal paste, cable ties, multimeter.
  • Safety precautions: power off, discharge static, work on non‑conductive surface.

Working inside a computer (assembling a desktop)

  • Preparation: gather tools, open case, ground yourself.
  • Installation order: power supply → motherboard → CPU + cooler → RAM → storage drives → graphics card → connect cables.
  • Cable management: improve airflow and accessibility.
  • First boot: check POST (Power‑On Self‑Test), enter BIOS/UEFI.

Introducing Windows operating system

  • Windows editions: Home, Pro, Enterprise – key differences.
  • User interface: desktop, taskbar, start menu, file explorer.
  • System tools: control panel, settings, task manager, device manager.
  • File systems: NTFS, FAT32, exFAT – features and compatibility.

All about motherboards

  • Form factors: ATX, microATX, Mini‑ITX – size and expansion.
  • Chipset: controls communication between CPU, RAM, peripherals.
  • Key connectors: CPU socket, RAM slots, PCIe slots, SATA ports, USB headers, power connectors.
  • BIOS/UEFI: firmware settings, boot order, hardware configuration.

Supporting processors and upgrading memory

  • CPU types: Intel (Core i3/i5/i7/i9) vs AMD (Ryzen).
  • Sockets: LGA (Intel) and PGA (AMD) – must match motherboard.
  • Cooling: air coolers, liquid cooling, thermal paste application.
  • RAM: DDR generations (DDR3, DDR4, DDR5), capacity, speed, dual‑channel configuration.
  • Upgrade procedure: check compatibility, insert firmly, enable XMP in BIOS.

Hard drives

  • HDD (Hard Disk Drive): magnetic platters, high capacity, slower, mechanical.
  • SSD (Solid State Drive): NAND flash, fast, durable, no moving parts.
  • Interfaces: SATA, NVMe (M.2) – speed differences.
  • Drive partitioning: MBR vs GPT, formatting.

Installing Windows

  • Installation media: USB drive (Rufus) or DVD with Windows ISO.
  • BIOS settings: set boot order to installation media.
  • Step‑by‑step: language, license, drive selection, partitioning, copying files.
  • Post‑installation: drivers, Windows Update, basic software.

I/O and Storage devices

  • Input devices: keyboard, mouse, scanner, microphone.
  • Output devices: monitor, printer, speakers.
  • External storage: USB flash drives, external HDD/SSD, memory cards.
  • Connectivity: USB (types A, C), HDMI, DisplayPort, Thunderbolt.

Maintaining Windows

  • Disk cleanup: remove temporary files, empty recycle bin.
  • Defragmentation (HDD only): optimize drive layout.
  • Updates: keep Windows and drivers current.
  • Backup: file history, system restore points, image backup.
  • Security: Windows Defender, firewall, avoid untrusted software.

Troubleshooting Windows and applications

  • Common issues: slow performance, blue screen (BSOD), application crashes.
  • Tools: event viewer, task manager, resource monitor, reliability monitor.
  • Steps: restart, check updates, run virus scan, safe mode, system restore.
  • Application‑specific: reinstall, compatibility mode, check logs.

Troubleshooting Hardware problems

  • No power: check cables, power supply, motherboard LEDs.
  • Beep codes: diagnose with POST beeps (refer to manual).
  • Overheating: clean fans, reapply thermal paste, check airflow.
  • Component failure: test with known‑good parts (swap method).
  • Tools: multimeter (PSU testing), loopback plugs.
