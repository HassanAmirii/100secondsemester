CSC106: Hardware Maintenance Cheat Sheet

components · assembly · Windows · motherboards · troubleshooting

1. Parts & Tools

  • Key components CPU, RAM, motherboard, PSU, storage (HDD/SSD), GPU, cooling fans.
  • Essential tools Phillips screwdriver (#0, #1), anti‑static wrist strap, thermal paste, multimeter, cable ties, flashlight.
  • Safety Power off, unplug, ground yourself, avoid magnetic tools near drives.

2. Desktop Assembly

  • Installation order PSU → motherboard (with CPU, RAM, cooler) → drives → expansion cards → connect front panel & power cables.
  • First boot Listen for beep, enter BIOS (DEL/F2), check CPU temp, RAM detection.

3. Windows OS basics

  • Editions Home (basic), Pro (bitlocker, remote desktop), Enterprise (orgs).
  • Key tools Task Manager (Ctrl+Shift+Esc), Device Manager, Disk Management, Event Viewer.
  • File systems NTFS (security, large files), FAT32 (compatibility, max 4GB file), exFAT (flash drives).

4. Motherboards

  • Form factors ATX (standard), microATX (smaller), Mini‑ITX (compact).
  • Chipset Intel: Z, B, H series; AMD: X, B, A series – determines features (overclocking, PCIe lanes).
  • BIOS/UEFI Firmware interface; UEFI supports mouse, secure boot, GPT disks.

5. CPU & RAM

  • CPU sockets Intel LGA (land grid array), AMD PGA (pin grid array) – must match motherboard.
  • Cooling Air cooler (heatsink+fan) or liquid (AIO). Always use thermal paste.
  • RAM DDR3, DDR4, DDR5 – not interchangeable. Dual‑channel: install in matched slots (usually slots 2 and 4).

6. Storage drives

  • HDD Magnetic platters, high capacity, slower, sensitive to shock.
  • SSD NAND flash, fast, durable, no moving parts. SATA SSD (~550 MB/s) vs NVMe M.2 (up to 7000 MB/s).
  • Partitions MBR (legacy, ≤2TB) and GPT (modern, >2TB, required for UEFI).

7. Windows installation

  • Media Create bootable USB with Windows ISO (Rufus or Media Creation Tool).
  • BIOS settings Boot order → USB first; disable secure boot if needed.
  • Steps Language → install now → license → custom (advanced) → partition drive → next.
  • Post‑install Install chipset drivers, GPU drivers, Windows updates.

8. I/O & external storage

  • Input devices Keyboard, mouse, touchpad, scanner, webcam, microphone.
  • Output devices Monitor, printer, speakers, headphones.
  • External storage USB flash drive, external HDD/SSD, memory card (SD, microSD).
  • Ports USB‑A, USB‑C, HDMI, DisplayPort, Thunderbolt, audio jacks.

9. Windows maintenance

  • Disk cleanup Remove temp files, recycle bin, old updates (cleanmgr).
  • Defrag/optimize HDD: defrag; SSD: TRIM (automatic, do not defrag SSD).
  • Backup File history, system restore points, full image backup (Windows Backup).
  • Security Keep Windows Update on, run Defender scans, avoid suspicious downloads.

10. Windows & app troubleshooting

  • Common issues Slow boot, blue screen (BSOD), app crashes, driver problems.
  • Tools Event Viewer (check logs), Task Manager (performance), Resource Monitor, Reliability Monitor.
  • Fixes Restart, run sfc /scannow, check updates, boot into safe mode, system restore.

11. Hardware troubleshooting

  • No power Check power cable, PSU switch, test outlet; try known‑good PSU.
  • Beep codes Refer to motherboard manual – e.g., 1 long + 2 short often means video issue.
  • Overheating Clean dust, verify fan operation, re‑apply thermal paste, improve airflow.
  • Component swap Replace with known‑good part to isolate fault (RAM, PSU, GPU).
