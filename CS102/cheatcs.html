COS102: Complete Cheat Sheet

problem‑solving · C · Visual Basic 6 — one‑page summary

1. Problem‑Solving Techniques

  • Program Development Cycle Analysis (define problem, I/O)
    Design (plan algorithm)
    Coding (implementation)
    Testing & Debugging
    Documentation
  • Algorithm essentials Definition: finite, unambiguous steps for a solution.
    Factors: time requirement (most important), definiteness, finiteness, effectiveness, generality.
    Examples: average of three numbers, factorial, prime testing.
  • Flowchart symbols (ISO) Oval: start/stop
    Parallelogram: input/output
    Rectangle: process (arithmetic, store)
    Diamond: decision (conditional)
    Flow lines, connectors (on‑page/off‑page)
  • Pseudocode English‑like description of logic – easy to read and modify. No strict syntax.
  • Other tools Hierarchy charts (HIPO) show module relationships.

2. C Programming Language

  • C fundamentals Creator: Dennis Ritchie (Bell Labs, 1970s).
    Features: simple, versatile, rich operators, fast, system programming.
    Data types: int, float, double, char.
  • Structure & I/O Statements end with ;
    Output: printf("format", vars);
    Variables & pointers: pointers store memory addresses.
  • Control flow – conditional if (condition) { … } else { … }
    else if ladder, nested if-else (comparison with ==)
  • Loops while (entry‑checked), do-while (executes at least once), for (initialization; condition; increment)
  • Other flow break (exit loop), continue (skip iteration), switch (multi‑branch), goto (unconditional jump)

3. Visual Basic 6 (VB6)

  • Core paradigm Object‑based, event‑driven — code executes in response to events (clicks, timer).
    RAD: rapid application development with GUI.
  • IDE components Toolbox, Properties window, Project Explorer, Code window, Form Layout, Immediate window (debugging), Object Browser.
  • Common controls Label, TextBox, CommandButton, OptionButton, CheckBox, Frame, ListBox, ComboBox, DriveListBox, DirListBox, FileListBox, Timer.
    Form is an object but not a control.
  • Key properties Visual: BackColor, ForeColor, Font, Caption, Text, Visible.
    Layout: Left, Top, Width, Height.
    Behavioural: Enabled, MousePointer, TabIndex.
  • Programming basics Event procedure: e.g., Command1_Click()
    Comment: apostrophe '
    Naming convention: txt for TextBox, cmd for CommandButton.
  • Design‑time vs. run‑time Properties can be set in the Properties window (design) or via code (run).
