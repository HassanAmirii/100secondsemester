MTH102: Complete Cheat Sheet

week-by-week formulas for calculus topics

Week 1: Function of a Real Variable

  • Function Notation y = f(x), domain D_f, range R_f.
  • Composition and Inverse (f o g)(x) = f(g(x)), f^(-1)(f(x)) = x.
  • Common Function Forms f(x) = ax + b, f(x) = ax^2 + bx + c, f(x) = 1/x, f(x) = |x|.

Week 2: Graphs of Functions

  • Quadratic Vertex For y = ax^2 + bx + c, vertex at x = -b/(2a).
  • Intercepts y-intercept: set x = 0; x-intercepts: solve f(x) = 0.
  • Graph Transformations y = f(x - h) + k, y = a f(x), y = f(bx).

Week 3: Limits of Functions

  • Limit Definition lim(x->c) f(x) = L.
  • Standard Limits lim(x->0) sin x / x = 1, lim(x->0) (1 - cos x)/x^2 = 1/2.
  • Algebra of Limits lim(f + g) = lim f + lim g, lim(fg) = (lim f)(lim g), lim(f/g) = (lim f)/(lim g).

Week 4: Continuity of Functions

  • Continuity at x = c f(c) exists, lim(x->c) f(x) exists, and lim(x->c) f(x) = f(c).
  • Continuity on Interval Continuous on [a,b] if continuous at each point in the interval.

Week 5: The Derivative as a Limit

  • First Principles f'(x) = lim(h->0) [f(x+h) - f(x)] / h.
  • Tangent Line At (x0, y0), slope m = f'(x0), tangent: y - y0 = m(x - x0).

Week 6: Techniques of Differentiation I

  • Basic Rules d/dx(c) = 0, d/dx(x^n) = n x^(n-1), d/dx[cf(x)] = c f'(x).
  • Linearity d/dx[u +/- v] = u' +/- v'.
  • Common Derivatives d/dx(sin x)=cos x, d/dx(cos x)=-sin x, d/dx(e^x)=e^x, d/dx(ln x)=1/x.

Week 7: Techniques of Differentiation II

  • Product Rule (uv)' = u'v + uv'.
  • Quotient Rule (u/v)' = (u'v - uv')/v^2.
  • Chain Rule d/dx[f(g(x))] = f'(g(x))g'(x).
  • Implicit Differentiation Differentiate both sides with respect to x and solve for dy/dx.

Week 8: Integration as the Inverse of Differentiation

  • Antiderivative If dF/dx = f(x), then integral f(x) dx = F(x) + C.
  • Power Rule integral x^n dx = x^(n+1)/(n+1) + C, n != -1.
  • Basic Integrals integral 1/x dx = ln|x| + C, integral e^x dx = e^x + C.

Week 9: Techniques of Integration

  • Substitution u=g(x) gives integral f(g(x))g'(x) dx = integral f(u) du.
  • Integration by Parts integral u dv = uv - integral v du.
  • Partial Fractions For rational functions, decompose first, then integrate each term.

Week 10: Definite Integrals and Applications

  • Fundamental Theorem integral_a^b f(x) dx = F(b) - F(a).
  • Area Under Curve A = integral_a^b f(x) dx for f(x) >= 0.
  • Area Between Curves A = integral_a^b [f(x) - g(x)] dx (top minus bottom).
