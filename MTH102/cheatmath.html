MTH102: Complete Cheat Sheet

calculus · algebra · vectors · complex numbers · differential equations

1. Mathematical Foundations

  • Number Systems Natural (ℕ), Integers (ℤ), Rational (ℚ), Irrational, Real (ℝ).
    Binary, octal, hexadecimal bases.
  • Set Operations Union (∪), Intersection (∩), Complement (A'), Symmetric difference (A ∆ B).
    Venn diagrams for visual representation.
  • Indices & Logarithms am · an = am+n, (am)n = amn
    loga(xy) = logax + logay, loga(x/y) = logax - logay
    logaxn = n logax
  • Quadratic Equation ax² + bx + c = 0
    Roots: x = [-b ± √(b² - 4ac)] / 2a
    Discriminant: Δ = b² - 4ac
    Sum of roots = -b/a, Product = c/a
  • Functions Domain, range, independent/dependent variables.
    Inverse: f⁻¹(f(x)) = x, composition: (f ∘ g)(x) = f(g(x))
  • Inequalities & Absolute Value |x| = { x if x ≥ 0, -x if x < 0 }
    |x - a| < b  ⇒  a - b < x < a + b

2. Core Calculus

  • Limits limx→c f(x) = L (approaches L)
    Standard limits: limx→0 sin x / x = 1, limx→0 (eˣ - 1)/x = 1
  • Differentiation dy/dx = gradient of tangent.
    Product rule: (uv)' = u'v + uv'
    Quotient rule: (u/v)' = (u'v - uv') / v²
    Chain rule: dy/dx = dy/du · du/dx
    Standard derivatives:
    d/dx (xⁿ) = n xⁿ⁻¹
    d/dx (sin x) = cos x
    d/dx (cos x) = -sin x
    d/dx (ln x) = 1/x
    d/dx (eˣ) = eˣ
  • Applications Tangent: y - y₀ = m (x - x₀)
    Stationary points: solve dy/dx = 0.
    Second derivative test: d²y/dx² > 0 → min, < 0 → max, = 0 → possible inflection.
  • Integration Indefinite: ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C (n ≠ -1)
    ∫ 1/x dx = ln|x| + C
    ∫ sin x dx = -cos x + C
    ∫ cos x dx = sin x + C
    ∫ eˣ dx = eˣ + C
    ∫ (ax + b)ⁿ dx = (ax + b)ⁿ⁺¹ / [a(n+1)] + C

3. Advanced Topics

  • Sequences & Series A.P.: aₙ = a + (n-1)d, Sₙ = n/2 [2a + (n-1)d]
    G.P.: aₙ = a rⁿ⁻¹, Sₙ = a(1 - rⁿ)/(1 - r) (r≠1)
    Sum to infinity (|r|<1): S∞ = a/(1 - r)
    Maclaurin series: f(x) = f(0) + f'(0)x + f''(0)x²/2! + ...
  • Vectors & Matrices Dot product: a·b = |a||b|cos θ
    Cross product: |a × b| = |a||b| sin θ
    2×2 determinant: |a b; c d| = ad - bc
    Matrix inverse (2×2): A⁻¹ = 1/(ad - bc) [d -b; -c a]
  • Complex Numbers j² = -1, j³ = -j, j⁴ = 1
    Polar form: z = r(cos θ + j sin θ)
    De Moivre: (cos θ + j sin θ)ⁿ = cos nθ + j sin nθ
  • Differential Equations 1st order separable: dy/dx = g(x)h(y) → ∫ dy/h(y) = ∫ g(x) dx
    Linear 1st order: dy/dx + P(x)y = Q(x), integrating factor μ = e∫ P dx
    2nd order homogeneous: ay'' + by' + cy = 0 – solve characteristic eqn.
  • Probability & Statistics Mean μ = Σx/n, variance σ² = Σ(x-μ)² / n
    Binomial: P(X=k) = C(n,k) pᵏ (1-p)ⁿ⁻ᵏ
    Poisson: P(X=k) = e⁻ˡ lᵏ/k!
    Normal: bell curve, parameters μ, σ.
